README.txt
==========

File Format: 
- ONLY .txt files.  Keep it pure ascii.  No binaries.  No .doc.  

File Names: 
- For meetings, use format "minute_2004_10_29.txt".  For agenda, use format "agenda_2004_10_29.txt".  Note that the year needs to be four digits, with month and date having two digits.

File Contents: 
- "minute" is obviously minutes for the meeting.  
- "agenda" is for anybody to edit before and leading up to the meeting, for adding topics to discuss.
